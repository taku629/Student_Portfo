# 🚀 Student Portfo - 学生投資家応援ポートフォリオ管理アプリ

「Student Portfo」は、学生投資家が自分の資産を楽しく、直感的に管理できるように設計されたウェブアプリケーションです。
証券アプリのスクリーンショットをAI（Gemini API）で解析し、一瞬でポートフォリオを構築します。

![Student Portfo Demo](https://via.placeholder.com/1200x600/060613/00ff9d?text=Student+Portfo+Demo)

## ✨ 主な機能

- **📸 AI資産スキャン**: 証券口座のスクリーンショットをアップロードするだけで、AIが銘柄・数量・価格を自動抽出。
- **🎮 ゲーミフィケーション**: 資産額や更新頻度に応じて「レベルアップ」や「称号」を獲得。投資をゲーム感覚で継続。
- **🎯 目標メーター**: 「MacBook購入」「卒業旅行」など、学生らしい目標への到達度をリアルタイム表示。
- **🔮 未来シミュレーター**: 現在の資産を複利で運用した場合の20年後の姿を、美しいグラフで予測。
- **📱 PWA対応**: スマホのホーム画面に追加して、ネイティブアプリのように利用可能。

## 🛠️ テクノロジー

- **Frontend**: HTML5, CSS3 (Glassmorphism Design), JavaScript (Vanilla)
- **AI/OCR**: Google Gemini 1.5 Flash (Generative AI)
- **Charts**: Chart.js
- **Animations**: Canvas-confetti, CSS Animations
- **PWA**: Service Worker, Web App Manifest

## 🚀 使い方

1. リポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開くか、GitHub Pagesなどでホストします。
3. 初回起動時に Gemini API Key を設定してください。
4. あとはスクショをアップロードするだけで、あなたの投資人生が可視化されます！
